# 🧠 KeenKeeper

## 📌 Project Overview

**KeenKeeper** is a modern React-based web application designed to help users manage and visualize data efficiently. It provides an interactive and user-friendly interface with smooth navigation and dynamic data handling.

---

## 🚀 Technologies Used

- ⚛️ React (v19)
- 🧭 React Router (v7)
- 🎨 Tailwind CSS (v4)
- 🌼 DaisyUI
- 📊 Recharts
- 🔔 React Toastify
- 🌐 Axios
- ⚡ Vite

---

## ✨ Key Features

### 1. Dynamic Routing System

- Built with React Router v7
- Supports nested routes and dynamic segments
- Smooth navigation across pages

### 2. Interactive Data Visualization

- Integrated with Recharts
- Displays charts and analytics in a clean UI
- Helps users understand data easily

### 3. Modern UI & UX

- Styled with Tailwind CSS and DaisyUI
- Fully responsive design
- Clean and minimal interface

---

## 📦 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/keenkeeper.git

# Go to project folder
cd keenkeeper

# Install dependencies
npm install

# Run the project
npm run dev
```
